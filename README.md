# Master Control - 自动补给模组

一个为 Minecraft 1.21.1 Fabric 设计的自动化模组，专用于自动补充钓鱼相关道具。<br>
需装前置### Fabric API

## 功能简介

- **双触发机制**：工具破损音效 或 聊天出现“用完了”后，等待“您获得”消息，自动执行补给。
- **智能选择**：每个道具可设置优先级（0-100），数值越大越优先，相同优先级随机选择。
- **全自动流程**：自动打开界面、检查槽位、按优先级选取道具、关闭界面、使用物品。
- **可视化配置**：按 F6 打开配置界面，分组管理所有道具，实时调整启用状态和优先级。

## 如何使用

2. **配置**：
    - 游戏中按 **F6** 打开配置界面。
    - 在“功能开关”组中开启需要的开关（换鱼饵/鱼线 或 换道具）。
    - 在各分组中将需要自动补给的道具设为“是”（绿色），并用 +/- 调整优先级。
3. **自动执行**：
    - 当工具破损（音效）或聊天出现“用完了”时，模组自动等待“您获得”消息。
    - 收到“您获得”后，自动按优先级选择道具并完成补给。

## 注意事项

- 本模组基于特定服务器界面布局设计（双箱子界面 + 玩家背包），若在其他环境使用需调整槽位索引。
- 默认快捷键为 F6，可在游戏控制设置中修改。<br>
